# BSS-Test Examples

This directory contains example scripts demonstrating various features of the BSS-Test framework.

## Quick Start

The simplest way to get started:

```bash
python examples/quick_start.py
```

This example demonstrates:
- Loading CWRU bearing data (or generating synthetic data if unavailable)
- Signal preprocessing
- CWT-based time-frequency analysis
- SOBI blind source separation
- Evaluation and visualization

## Examples List

### Basic Usage
- `quick_start.py` - Basic BSS pipeline example

### Advanced Usage
- `custom_tfa.py` - Using different TFA methods (CWT, STFT, WPT, VMD, EMD)
- `custom_bss.py` - Using different BSS algorithms (SOBI, FastICA, JADE, PICARD)
- `ml_pipeline.py` - Complete ML classification pipeline
- `cross_validation.py` - Cross-validation example

### Data Analysis
- `cwru_analysis.py` - Comprehensive CWRU dataset analysis
- `phm_analysis.py` - PHM 2010 milling tool wear analysis
- `nasa_analysis.py` - NASA milling dataset analysis

### Utilities
- `feature_extraction.py` - Feature extraction examples
- `visualization.py` - Custom visualization examples
- `config_example.py` - Configuration management examples

## Running Examples

All examples can be run from the project root directory:

```bash
# From project root
python examples/quick_start.py

# Or with specific options
python examples/cwru_analysis.py --fault-type inner_race_007
```

## Data Requirements

Some examples require downloaded datasets:

- **CWRU**: Download from https://zenodo.org/records/10987113 and place in `data/cwru/`
- **PHM 2010**: Download from PHM Society and place in `data/phm2010_milling/`
- **NASA**: Download mill.mat and place in `data/phm2010_milling/`

If data is not available, examples will use synthetic data generated by `src/utils.py`.

## Output

All examples save results to `outputs/examples/` directory with descriptive subdirectories.
